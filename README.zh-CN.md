# Skills

[English](README.md)

锻炼 AI 取代不了的那部分能力。

有了汽车之后，人们依然会去健身房练腿——不是因为汽车不够快，就像**思考本身不该被替代**。AI 是放大器，也是磨刀石。它放大你已有的能力，磨砺你愿意花时间打的基础。你的能力越强，AI 的效果就越好——**两者交替螺旋上升**。

**这个仓库关注的不是工具，是人。**

市面上大多数 AI skill 在教你怎么写更好的 prompt、跑更快的脚本、做出更顺滑的工作流。这些有用，但它们会自己积累且变化太快。

真正难积累的是另一件事：**判断力**。知道什么值得做、什么该放弃；什么是用户嘴上说要但不会付钱的需求；怎么决定团队里反复纠结的方向问题。这种能力 AI 替不了你练，只有你自己在反复做决策、收到反馈、修正判断中长出来。

我希望可以通过这些skills，提高我认为的工程师应该具备的重要能力。

大部分skills源自我个人的经验和想法，也有可能来自其他地方的灵感优化后的成果（我会注明来源）。
